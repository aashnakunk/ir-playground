jazzbot

An interactive playground for IR & RAG, themed in jazz.

tfidf bm25 semantic hybrid hnsw rag cag

how this works

Each page below explores one retrieval or generation technique using a small jazz corpus (~30 docs). Every page has a live visual and a sidebar tutor (OpenAI) that can see exactly what you're looking at — ask it "why did doc #10 win?" and it will reference real scores.

topics

TF-IDF

Per-term contributions and a term×doc heatmap.

BM25

TF-IDF with saturation and length normalization.

Semantic search

Cosine over embeddings + 2D map of meaning.

Hybrid (RRF)

BM25 + semantic, fused without score calibration.

HNSW

Layered graph, search traversal, M / ef sliders.

RAG

Retrieve top-k → prompt → generate. Every step visible.

CAG

Full corpus in cached system prompt. Compare cost vs RAG.
