Hybrid search

Combine keyword search and meaning search. Usually beats either one alone.

try this

  1. Search something with a specific name (a person, a place, a dish). Keyword search (BM25) usually wins.
  2. Search something conceptual ("mood of late-night music", "tangy fermented stuff"). Meaning search (Semantic) usually wins.
  3. The right column is the fused result. It picks docs that did well in either list.
  4. Notice the small print under each fused result — it shows where the doc ranked in BM25 and Semantic separately.
  5. Ask the tutor: "which docs were ranked high in one but not the other?"
what's RRF?

Reciprocal Rank Fusion. For each doc, take 1/(60+rank) from each list and add them up. Doesn't care about raw scores — just ranks — which is why it works even when the two systems' scores aren't on the same scale.
